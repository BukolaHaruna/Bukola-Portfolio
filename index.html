  • Import Datasets from CSV or APIs.
  • Data Transformation by Filtering & Sorting Data using WHERE, ORDER BY & LIMIT:
    • Aggregating data where needed using GROUP BY with functions like SUM(), AVG(), COUNT(), MIN(), MAX().
  • Exploratory Data Analysis (EDA) using Pivot Tables & Charts:

  • Advanced SQL Techniques:

Dashboards using Power BI / Tableau

In Power BI, you can either clean your data in Excel before importing it or clean it directly in Power BI using the Transform Data option in Power Query. Cleaning data in Power BI allows for more flexibility and efficiency, especially for large datasets. In this case, we'll assume the datasets have already been cleaned in Power BI, so the next steps will focus on;

  • Creating DAX Measures & Calculated Columns to define key metrics.
  • Reducing data size by removing unnecessary columns & tables.
  • Building Visualisations & Dashboard:
    • Selecting appropriate Charts like bar, line, pie, and KPI visuals.
    • Adding Slicers & Filters allowing users to interact with data.
    • Using Tooltips to provide additional insights without cluttering the dashboard.
  • Publish & Share Reports.
